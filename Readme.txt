1. Brief description

  Actual Transparent Window lets you apply transparency effect to
    any window in Windows XP/2000/2003/Vista/7.
    The transparency effect looks very impressive and may be useful
    with such objects as Windows Taskbar, Winamp media player,
    drop-down menus and other windows.
    Actual Transparent Window lets you set individual transparency
    rate for each window. You can change transparency from 0%
    (solid) to 100% (fully transparent) to achieve the desired effect.

2. Installation/Uninstallation

     Installation

       If you've downloaded the self-extracting installation package named
       like "atwsetup.exe" just execute it and follow the setup
       instructions.

       If you've downloaded the compressed archive file named like
       "atwsetup.zip" unpack it to the temporary folder and execute
       the setup.exe file.

       After the succesful installation run the program from new
       "Actual Transparent Window" program group.

     Uninstallation

       There are two ways to uninstall Actual Transparent Window.

       Control Panel:
         - Open the Windows Control Panel.
         - Double click Add/Remove Programs.
         - Double click the Actual Transparent Window list box entry.
         - Follow the instructions in the Uninstall Wizard.

       Start Menu:
         - Click the Start button.
         - Select All Programs.
         - Select Actual Transparent Window.
         - Click Uninstall Actual Transparent Window.
         - Follow the instructions in the Uninstall Wizard.
